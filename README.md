# helm-kubectl-aws
helm + kubectl + awscli docker helper

```bash
$ docker run -it \
    -e AWS_ACCESS_KEY_ID="[YOUR_AWS_ACCESS_KEY_ID]" \
    -e AWS_SECRET_ACCESS_KEY="[YOUR_AWS_SECRET_ACCESS_KEY]" \
    -e AWS_DEFAULT_REGION="[YOUR_AWS_AWS_DEFAULT_REGION]" \
    -e AWS_ACCESS_KEY_ID="[YOUR_AWS_AWS_ACCESS_KEY_ID]" \
    teochenglim/helm-kubectl-aws bash

$ docker run -it \
    -e AWS_ACCESS_KEY_ID="[YOUR_AWS_ACCESS_KEY_ID]" \
    -e AWS_SECRET_ACCESS_KEY="[YOUR_AWS_SECRET_ACCESS_KEY]" \
    -e AWS_DEFAULT_REGION="[YOUR_AWS_AWS_DEFAULT_REGION]" \
    -e AWS_ACCESS_KEY_ID="[YOUR_AWS_AWS_ACCESS_KEY_ID]" \
    teochenglim/helm-kubectl-aws ash
```
